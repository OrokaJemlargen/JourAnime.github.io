Politique de confidentialité

Dernière mise à jour : décembre 2025

JourAnime est un outil interne utilisé par un seul créateur pour gérer et publier ses propres vidéos sur TikTok et d’autres plateformes.

1. Données collectées

L’application ne collecte aucune donnée personnelle concernant d’autres utilisateurs TikTok.

2. Données traitées

JourAnime peut traiter uniquement :

  • des informations techniques nécessaires à la publication des vidéos,
  • des jetons d’authentification fournis par l’API officielle TikTok pour le compte du créateur.

3. Utilisation des données

Les données sont utilisées exclusivement pour :

  • envoyer les vidéos sur le compte TikTok du créateur,
  • assurer le bon fonctionnement technique de l’outil.

4. Partage des données

Aucune donnée n’est vendue, partagée ou transmise à des tiers. Les échanges se font uniquement avec les plateformes concernées via leurs APIs officielles.

5. Stockage

Les données techniques et jetons d’accès peuvent être stockés localement sur la machine du créateur ou sur une infrastructure utilisée par le créateur.
